## 📂 Organização do Repositório

### 🌐 Programação Web (Front-end)
Projetos focados na criação de interfaces modernas, responsividade e experiência do usuário (UX).

* **[Food Panda - Escola de Confeitaria](./Programação%20Web/AV1%20-%20Site%20Food%20Panda):** Landing page profissional para uma escola de gastronomia. 
    * **Destaque:** Uso de Flexbox/Grid para garantir responsividade em dispositivos móveis.
    * *Techs:* HTML5 e CSS3.

### ☕ Desenvolvimento Java (Back-end & Desktop)
Aplicações focadas em lógica de negócio, Programação Orientada a Objetos (POO) e integração com banco de dados.

* **[Projeto Psystem](./Projeto-Psystem):** Sistema de recomendação de treinos para estúdios de Pilates.
    * **Destaque:** Implementação de regras de negócio complexas e persistência de dados.
    * *Techs:* Java e SQL (MySQL).

* **[Projeto Auto Agenda](./Projeto-Auto-Agenda):** Sistema de gerenciamento para oficinas mecânicas com foco em agendamentos e estoque.
    * **Destaque:** Sistema robusto com autenticação obrigatória, controle de níveis de acesso e dashboards para gestão de serviços e colaboradores.
    * *Techs:* Java, HTML5, CSS3, Bootstrap, JavaScript e banco de dados H2.
