# 🏆 Simulador de Campeonato de Futebol em Java

Projeto desenvolvido para a disciplina de **Algoritmos e Programação I**, com o objetivo de aplicar conceitos de Java moderno, como `record`, coleções e lógica de negócios.

---

## 📌 Descrição

Este projeto simula partidas de futebol entre times, calcula estatísticas (pontos, vitórias, empates, derrotas, saldo de gols) e classifica os times em categorias de desempenho.

---

## 🚀 Funcionalidades

- ✔ Processamento de partidas a partir de strings
- ✔ Cálculo automático de:
  - Pontos
  - Vitórias, empates e derrotas
  - Gols pró e contra
  - Saldo de gols
- ✔ Classificação dos times por categoria:
  - 🥇 Líder
  - 🟢 G4 (Zona de classificação)
  - 🟡 Meio de tabela
  - ⚠️ Alerta
  - 🔴 Rebaixamento
- ✔ Uso de `record` (imutabilidade)
- ✔ Estruturas de repetição (`for` e `while`)

---

## 🧠 Regras de Pontuação

- Vitória → **3 pontos**
- Empate → **1 ponto**
- Derrota → **0 pontos**

---

## 📊 Categorias de Classificação

| Pontos   | Categoria                         |
|----------|----------------------------------|
| 20+      | 🥇 Líder                         |
| 14 a 19  | 🟢 G4 - Zona de Classificação    |
| 8 a 13   | 🟡 Meio de Tabela                |
| 4 a 7    | ⚠️ Alerta                        |
| 0 a 3    | 🔴 Rebaixamento                  |

---

## 🛠️ Tecnologias Utilizadas

- ☕ Java (JDK 17+ recomendado)
- 📦 Collections (`List`, `ArrayList`)
- 🧱 Programação Orientada a Objetos
- 🆕 `record` (Java moderno)
