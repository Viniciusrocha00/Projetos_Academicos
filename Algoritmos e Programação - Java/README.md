# Algoritmos e Programação Java I ☕

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-green?style=for-the-badge)

Este repositório centraliza a resolução das listas de atividades e exercícios práticos desenvolvidos em sala de aula para a disciplina de **Algoritmos e Programação Java**.

O objetivo deste repositório é documentar minha curva de aprendizado na linguagem Java, partindo dos conceitos fundamentais de lógica de programação até a implementação de estruturas mais complexas.

---


## 🚀 Tecnologias e Ferramentas

* **Linguagem:** Java (JDK 17 ou superior)
* **IDE utilizada:** [IntelliJ / Eclipse]
