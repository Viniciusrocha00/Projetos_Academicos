# Psystem - Recomendação de Treinos 🧘‍♂️

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)
![MySQL](https://img.shields.io/badge/mysql-%2300f.svg?style=for-the-badge&logo=mysql&logoColor=white)

Nesta seção, apresento o **Psystem**, um sistema desenvolvido como projeto acadêmico para um estúdio de Pilates. O foco principal é a automação da recomendação de treinos personalizados para os alunos, otimizando o acompanhamento instrutor-aluno.

O objetivo do projeto foi integrar lógica de programação robusta com persistência de dados eficiente:

* **Java:** Implementação de toda a lógica de negócio, POO (Programação Orientada a Objetos) e interface com o usuário.
* **SQL (MySQL/PostgreSQL):** Modelagem de dados e armazenamento estruturado de informações de alunos, exercícios e cronogramas.
* **Gestão de Treinos:** Algoritmo para recomendação de séries baseado no perfil e evolução individual de cada praticante.

---

## 🚀 Funcionalidades Principais

* **Cadastro de Alunos:** Registro completo de anamnese e restrições físicas.
* **Banco de Exercícios:** Catálogo detalhado de movimentos de Pilates com níveis de dificuldade.
* **Gerador de Treinos:** Sistema inteligente que sugere treinos com base no histórico do aluno.
* **Persistência:** Histórico de evoluções salvo de forma segura via SQL.

## 🛠️ Tecnologias Utilizadas


* **Linguagem:** Java (JDK 11+)
* **Banco de Dados:** SQL (JDBC para conexão)
* **Arquitetura:** MVC (Model-View-Controller) para organização de código.
