Nesta seção, apresento projetos desenvolvidos para a disciplina de Programação Web, focados na criação de interfaces responsivas, semânticas e interativas. O objetivo aqui foi dominar a tríade fundamental da web:

HTML5: Estruturação de conteúdo de forma semântica e acessível.

CSS3: Estilização avançada, layouts com Flexbox/Grid e design responsivo.

JavaScript (ES6+): Implementação de lógica de programação, manipulação do DOM e consumo de APIs.
