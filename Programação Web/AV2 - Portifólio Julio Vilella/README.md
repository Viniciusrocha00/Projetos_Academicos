Nesta seção, apresento o meu **Web Portfólio Pessoal**, um projeto desenvolvido para consolidar minha presença digital e exibir meus projetos acadêmicos e profissionais. O site funciona como uma central de informações, conectando recrutadores e parceiros às minhas competências em tecnologia.

O foco principal foi a criação de uma página institucional clean, priorizando a experiência do usuário e a clareza na exposição do perfil técnico:

* **Arquitetura Semântica:** Uso de tags HTML5 (header, nav, section, footer) para garantir acessibilidade e melhor indexação.
* **Identidade Visual:** Interface moderna utilizando a fonte *Poppins* e uma paleta de cores profissional (Azul e Branco).
* **Seções Modulares:** Organização estratégica entre "Sobre Mim", "Habilidades" e "Contato".
* **Integração de Mídia:** Inclusão de fotografia profissional e ícones sociais para aumentar o engajamento e confiabilidade.

---

## 📚 Estrutura do Site

O portfólio está organizado em quatro pilares fundamentais para a apresentação profissional:

1.  **Header Dinâmico:** Apresentação principal com foto de perfil, nome e cargo atual.
2.  **Sobre Mim:** Resumo da trajetória acadêmica em Sistemas de Informação e áreas de interesse (Web, Dados e IA).
3.  **Habilidades Técnicas:** Exposição visual das tecnologias dominadas (HTML, CSS, JS, Python, SQL e Git).
4.  **Central de Contato:** Formulário direto para mensagens e links rápidos para LinkedIn, GitHub e Instagram.

## 🚀 Funcionalidades Implementadas

* **Navegação por Âncoras:** Menu interativo que direciona o usuário suavemente para as seções de interesse.
* **Layout Responsivo:** Design adaptável para diferentes tamanhos de tela e dispositivos.
* **Captação de Mensagens:** Interface de formulário estruturada para recebimento de propostas e contatos.
* **Central de Redes Sociais:** Botões estilizados com ícones para acesso rápido ao ecossistema profissional do autor.

## 🛠️ Tecnologias Utilizadas

* **Estruturação:** HTML5
* **Estilização:** CSS3 (Custom fonts via Google Fonts e layouts baseados em containers).
* **Interatividade:** JavaScript (Lógica de navegação e comportamento da página).
* **Organização de Arquivos:** Estrutura profissional de pastas (assets/imagens, js, css).
