# Food Panda - Escola de Confeitaria 🐼🧁

![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

Nesta seção, apresento o projeto **Food Panda**, uma Landing Page desenvolvida para a disciplina de Programação Web (AV1). O projeto consiste em um site para uma escola de confeitaria que guia o aluno "do básico ao profissional".

O foco principal foi a construção de uma interface limpa, com alta legibilidade e foco em conversão (inscrição):

* **HTML5 Semântico:** Uso de tags estruturais para organizar o cabeçalho, seções de conteúdo e formulários.
* **Design Visual com CSS3:** Implementação de uma paleta de cores sóbria (azul marinho e creme) e tipografia moderna.
* **Componentização de Cards:** Criação de seções modulares para apresentar a ementa do curso.
* **Formulários Funcionais:** Estilização de uma seção de captura de leads (inscrição) com validação visual.

---

## 📚 Estrutura do Curso

O site apresenta uma trilha de aprendizado dividida em três pilares principais:

1.  **Módulo 1 - Confeitaria Profissional:** Focado em bases da confeitaria, mise en place e organização.
2.  **Módulo 2 - Cafeteria:** Introdução ao universo dos cafés especiais e técnicas de texturização de leite.
3.  **Módulo 3 - Massas:** Preparo de massas frescas e artesanais, do grão ao molho.

## 🚀 Funcionalidades Implementadas

* **Landing Page Responsiva:** Interface adaptável para diferentes dispositivos.
* **Navegação Âncora:** Links no menu que direcionam o usuário suavemente para as seções de Conteúdo e Inscrição.
* **Seção Hero:** Destaque principal com Call to Action (CTA) para engajamento imediato.
* **Formulário de Inscrição:** Interface clara para coleta de nome, e-mail e telefone.

## 🛠️ Tecnologias Utilizadas

* **Estruturação:** HTML5
* **Estilização:** CSS3 (Flexbox para o layout dos módulos e formulário)
* **Assets:** Imagens de alta qualidade integradas ao layout para reforçar a temática gastronômica.
